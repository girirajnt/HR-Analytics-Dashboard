# HR-Analytics-Dashboard
An interactive HR Analytics Dashboard built in Power BI to analyse employee attrition ,workforce demographics, job satisfaction, education level, and key HR KPIs.  It is designed to support HR teams in making data-driven decisions regarding workforce management, retention strategies, and organizational planning.

## Dashboard Features
Employee Count 
Attrition Count & Attrition Rate 
Active Empolyee Count 
Dipartment-wise Attrition Analysis
attrition by Gender 
Employee Distribution by Age Group 
Job Stisfaction Analysis 
Education field-wise Attrition 
Attrition Rate by Gender Across Different Age Group 
Interactive Degree Filers
